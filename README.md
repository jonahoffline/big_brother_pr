# TrafficCamsPR

A Ruby client and library for ITS DTOP's Traffic Cameras API.

## Installation
    gem install traffic_cams_pr

## Usage

### Basic Example Code
```ruby
require 'traffic_cams_pr'

client = TrafficCamsPR.new

client.all_cameras
```
